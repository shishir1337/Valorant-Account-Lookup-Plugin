== Description ==
The Valorant Account Lookup plugin allows you to display a Valorant account lookup form on product pages in WooCommerce. It also adds a custom field for users to enter their Valorant account details.

== Installation ==
1. Upload the `valorant-account-lookup` folder to the `/wp-content/plugins/` directory or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.

== Usage ==
To display the account lookup form and custom field, use the `[account_lookup]` shortcode on the desired page or post.

== Frequently Asked Questions ==
1. How can I display the account lookup form?
   Use the `[account_lookup]` shortcode on the desired page or post to display the form.

2. Where will the custom field be displayed?
   The custom field will be displayed on the product page in WooCommerce.
